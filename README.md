## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

This project runs in Node v15.12.0 engine and npm 7.6.3
Dependencies are the following:

evelopment:<br />

Cheerio @ 1.0.0<br />
Request @ 2.88.2<br />
Resquest-promise @ 4.2.6<br />

Install them all with the following command:

Development dependencies

```
npm i cheerio request resquest-promise
```

Install the packages named above, then, start the server running:

```
npm run dev
```
